# Ethereum-Ropsten-ToDo-List
A simple To-Do List dApps with Moralis SDK on Ropsten Ethereum Blockchain

## Demo
[ethereum-roptsen-to-do-list.on.fleek.co](https://ethereum-roptsen-to-do-list.on.fleek.co/)

## Usage
* Read messages
* Write a message
* Delete a message
